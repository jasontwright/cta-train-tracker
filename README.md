# Purpose

Just an area to play around with the [CTA Train Tracker API](http://www.transitchicago.com/developers/traintracker.aspx)

# Current Use Case

Access the [CTA Train Tracker API](http://www.transitchicago.com/developers/traintracker.aspx) and fetch information about the Sheridan Red Line stop and write that information to a file 'Sheridan.xml'.
